# SuperStore-Analytics-Dashboard

Superstore Analytics Report

Project Title: Multi-Dimensional Retail Analytics Report (Tableau)

Technical Stack: Tableau Desktop, Advanced Parameter Actions, Custom Calculated Fields.

Key Technical Features:

 - Dual-KPI Dynamic Engine: Engineered a unified dashboard interface using Parameters and Calculated Fields 
to toggle the entire visual workspace between Profitability and Volume metrics without increasing dashboard load time.

 - Geospatial Intelligence: Developed a geographic heat map with custom tooltips to identify "Profit Deserts" and high-growth states, enabling targeted regional marketing insights.

 - Advanced Time-Series Analysis: Implemented Moving Averages and Trend Lines to smooth out seasonal volatility, allowing for more accurate forecasting of future inventory needs.

 - Action Filters & Interactivity: Built a seamless UX where selecting a region on the map drills down into specific Sub-Category performance and specific Product-level profitability.

 - Product Affinity: Isolated the Canon imageCLASS 2200 as a primary profit driver, contrasting it against loss-leading furniture categories to recommend portfolio rebalancing.

Key Metrics:

 - Total Revenue: $2.30M | Operating Margin: 12.47%

 - Customer Retention: Analyzed via return rates (5.91% avg) to identify quality control issues.
